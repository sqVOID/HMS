🗄️ Database Setup

ℹ️ Setup Required:
The reports table needs additional columns to store food and item charges. Follow the steps below to complete the setup.

1 Quick Setup (Recommended)

Run the quick setup script which will automatically create users and add the missing database columns:

  • ✓ Creates user accounts (admin, staff, user)
  • ✓ Adds additional_food column to reports table
  • ✓ Adds additional_items column to reports table

Default Login: username: admin | password: password123

2 Add Database Columns Only

If you already have the system set up and just need to add the missing columns to the reports table:

  • ✓ Adds additional_food column
  • ✓ Adds additional_items column
  • ✓ Verifies columns in database
✓ What These Columns Do:
• additional_food - Stores food items added to bookings (e.g., "1 HOTDOG = ₱120.00")
• additional_items - Stores inventory items added to bookings (e.g., "1 Cover = ₱150.00")

✓ After Setup Complete

Once the setup is complete, you can:

  • ✓ Log in to the system with your credentials
  • ✓ Edit bookings and add additional charges
  • ✓ View reports with food and item charges
